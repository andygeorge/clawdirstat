## Project

`clawdirstat` is a Rust CLI tool to visualize disk usage, similar to KDirStat/WinDirStat.

## Commands

Once the Rust project is initialized:

```sh
cargo build          # build
cargo run            # run
cargo test           # all tests
cargo test <name>    # single test
cargo clippy         # lint
cargo fmt            # format
```
